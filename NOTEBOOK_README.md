# Jupyter Notebook: DFA vs Bonds Analysis

Интерактивный ноутбук для сравнительного анализа ЦФА и облигаций.

## 🚀 Как запустить

### 1. Активируй виртуальное окружение

```bash
source venv/bin/activate
```

### 2. Запусти Jupyter

```bash
jupyter notebook dfa_vs_bonds_analysis.ipynb
```

Или через JupyterLab (если установлен):

```bash
jupyter lab dfa_vs_bonds_analysis.ipynb
```

## 📓 Структура ноутбука

Ноутбук содержит **15 ячеек** с полным анализом:

### 📋 Подготовка (ячейки 1-2)
- Импорт библиотек
- Настройка окружения
- Параметры ЦФА

### 🔍 Анализ ЦФА (ячейки 3-4)
- Расчет доходности
- Купонные платежи
- График платежей

### 🏦 Анализ облигаций (ячейки 5-7)
- Подбор сопоставимых выпусков
- Статистика по выборке
- Детальный анализ

### ⚖️ Сравнительный анализ (ячейки 8-14)
- Сравнение доходности
- График спреда
- Профиль риск-доходность
- Декомпозиция спреда
- Рекомендации

### ✅ Заключение (ячейка 15)
- Итоговые выводы
- Рекомендации для разных инвесторов

## 📊 Ключевые графики

Ноутбук генерирует **4 интерактивных графика**:

1. **Купонные платежи** — 2 графика в одной ячейке:
   - Бар чарт купонов по периодам
   - Линия накопленного дохода

2. **Сравнение доходности** — bar chart:
   - ЦФА vs все облигации
   - Линия среднего значения
   - Аннотация спреда

3. **Анализ спреда** — horizontal bar plot:
   - Спред по каждой облигации
   - Цветовая кодировка (ЦФА выше/ниже)
   - Значения спредов

4. **Профиль риск-доходность** — scatter plot:
   - ЦФА, Облигации, Безрисковая ставка
   - Sharpe ratio для каждого инструмента
   - Линии от безрисковой ставки

5. **Декомпозиция спреда** — horizontal bar plot:
   - Факторы спреда
   - Вклад каждого фактора
