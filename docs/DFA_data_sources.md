## Что конкретно искать:

### Параметры выпуска ЦФА:

| Параметр | Что искать | Пример |
|----------|------------|--------|
| **Название** | "ЦФА Сбера-Х" | Сбер-01, Сбер-2024-1 |
| **Дата выпуска** | "выпущен" + дата | Сентябрь 2024 |
| **Объем выпуска** | "объем выпуска" + сумма | 5 млрд руб |
| **Срок обращения** | "срок" + месяцы/год | 6 месяцев, 1 год |
| **Доходность** | "доходность"/"купон" | 15% годовых |
| **Мин. сумма** | "мин. инвестиция"/"номинал" | 100 000 руб |
| **Периодичность** | "выплата купона" | Ежемесячно, в конце срока |
| **Оферта** | "досрочное погашение" | Через 3 месяца |

## План действий (прямо сейчас):

### Шаг 1: Найти выпуск (5-10 минут)

1. **Открой**: https://sberbank.ru
2. **Поиск по сайту**: введи "ЦФА" или "цифровой финансовый актив"
3. **Или**: https://rbc.ru → поиск "Сбер выпустил ЦФА"
4. **Найди**: пресс-релиз или страницу с параметрами выпуска

- **Мин. сумма: ___________ руб
- **Купон: ___________ (фиксированный/плавающий)
