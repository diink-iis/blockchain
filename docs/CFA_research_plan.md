## 4. Факторы для объяснения разницы

### А. Структурные факторы:
- **Ликвидность**: облигации торгуются на бирже → выше ликвидность → ниже yield
- **Инвесторская база**: ЦФА = qualified investors только → меньший спрос → выше yield
- **Налоги**: возможно разные режимы

### Б. Рисковые факторы:
- **Credit risk**: одинаковый (один эмитент)
- **Regulatory risk**: ЦФА - новый инструмент → неопределенность
- **Technology risk**: платформа может быть меньше протестирована

### В. Рыночные факторы:
- **Маркетинг**: ЦФА позиционируются как "инновационный" продукт
- **Доступность**: облигации доступны через любого брокера, ЦФА через спецплатформу
